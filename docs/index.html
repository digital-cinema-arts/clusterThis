clusterThis mantra geometry instancer

The clusterThis mantra geometry instancing procedural DSO/OTL will instance geometry to incoming points and will process the point attributes in a number of ways for the instanced geometry. Delays geomtery instantiation until render time thus, reduces size of IFD file and network bandwidth, allows access to render time data, add additional detail to scene.

Features:

  • Instance primitives at render time (sphere, cube, grid, tube, circle, curve, metaball)
  • Instance disk file (and apply CVEX deformation and attribute processing)
  • Open VDB post processing of instanced geometry and it's attributes
  • Motion blur
  • CVEX processing of instanced geometry
  • Cache file stores data when generating deep shadow maps to optimize instancing.
